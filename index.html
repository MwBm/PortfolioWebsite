AWSxINRIX Hackathon

My team and I designed and developed a Google Chrome extension using Claude 3 Sonnet via the AWS Bedrock API to improve website navigation. The extension reduced average navigation time by 15 seconds compared to the baseline performance of my teammates. I led the API integration for this project, using Python (Flask) to build the backend and Postman for testing. The workflow involved sending user input from the Chrome extension to a Flask server, which then passed the data to the Claude 3 Sonnet model through the AWS Bedrock API. I utilized prompt engineering to optimize the responses from Claude, which were then returned to the Chrome extension for user interaction. This setup provided a seamless and efficient browsing experience for users.

Disneyland Data Analysis

Using a Kaggle dataset, I analyzed the relationship between temperature and wait times at Disneyland. After cleaning the data with Pandas, I conducted a Pearson correlation test (revealing a weak linear relationship, r=0.16, p < .01) and segmented the data into temperature ranges for deeper insights. A Kruskal-Wallis H test confirmed significant differences in wait times across these categories (H=187, p < 0.01), showing temperature’s impact on ride waits. To predict wait times, I trained a RandomForestRegression model, achieving an R-squared value of .58, outperforming the baseline LinearRegression model's 0.28. While the model still needs refinement, the project highlights actionable trends and opportunities for improvement.
